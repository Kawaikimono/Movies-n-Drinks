# Cinnema & Sips

## Description
This site can randomly choose movies and drinks! You can pick out a genre you want to watch.
When you click "Yes!" on pick a movie to watch, you'll then given genres to choose from. After getting random movie, you'll be able to choose to have a alcohol or nonalcohol drink!


## User Story
AS AN user with curiosity or with no time to plan
I WANT to find random movies and random drinks to go along with
SO THAT I don't have to plan out ideas and enjoy my time
